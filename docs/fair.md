# FAIR principles

## Findable

- STAC item generated at `metadata/stac_item.json`.
- Spatial bbox and stable prototype identifier.
- Manifest in the offline bundle.

## Accessible

- Offline bundle is ZIP-packaged.
- All current prototype products are open files: CSV, GeoJSON, JSON, SVG, PNG, PDF, and ZIP.
- High-quality source downloads and omissions are recorded in metadata.

## Interoperable

- EPSG:4326 coordinates in prototype layers.
- Signal K resources export.
- OpenCPN supplemental GeoJSON package.
- Local SVG symbol assets, including OpenBridge icons when configured.
- Roadmap for S-57, KAP, MBTiles, GeoPackage, GeoTIFF, NetCDF, and Zarr.

## Reusable

- Provenance JSON for each source family.
- QC report for bathymetry.
- License notes for ODbL-derived data, OpenBridge icon assets, MPA sources, and bathymetry providers.
- Source quality and freshness manifests for high-quality runs.
- Warning that generated charts are experimental and not for navigation.
